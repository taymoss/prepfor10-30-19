Scales and Why the Are Useful

They are functions whose parameters you define. A scale is a mathematical relationship and has no visual output directly.Scales come up with domains and ranges so that graphs don't become overly large. It is useful to visualization because it makes the graphs or whatever you are presenting smaller instead of when new data is added growing exponentially larger. Scales help balance the data to make it more presentable to look at, instead of making the graph ever growing. Also, it helps order the data so that it is easier for the user to read and interpret the graph.
